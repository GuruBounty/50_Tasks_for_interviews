#### Задача 6: Буферизированная очередь задач (worker-pool)
**Краткое описание**  
Нам дано: список задач — это целые числа `x`. Есть N воркеров и канал-очередь задач **с буфером ровно 10**.  
Нужно: обработать **все** задачи в worker-pool и вывести **сумму результатов**.  
Чтобы сделать — вспомни: `goroutine`, `chan`, `WaitGroup`, закрытие каналов и сбор результатов без гонок.  
**Цель навыка**
- собрать корректный worker-pool;
- правильно закрывать каналы и завершать воркеры;
- аккуратно суммировать результаты (без data race);
- не зависнуть на чтении/записи из каналов.
 
**Условие**  
**Нам дано:**

- `M` задач (числа),
- `N` воркеров,
- канал задач размером **10**.

Напиши программу, которая:
1. отправит все M задач в буферизированный канал задач (capacity = 10)
2. запустит N воркеров, каждый считает `f(x) = x * x`
3. соберёт все результаты и выведет их сумму.

При этом **нельзя**:
- терять задачи,
- оставлять горутины висеть,
- писать в закрытый канал,
- суммировать из нескольких горутин в одну переменную без синхронизации.

**Нужно**:
- корректно закрыть канал задач,
- корректно дождаться воркеров,
- корректно завершить сбор результатов.
