#### Задача 4: Распределённые вычисления (Map-Reduce)
Нам дано:
- Срез целых чисел

Напиши программу, которая:
1. Делит входные данные на `3` равные части (чанка)
2. Запускает `3` map-горутины (по одной на чанк), каждая из которых:
   - Вычисляет квадраты чисел в своем чанке
   - Вычисляет сумму квадратов в чанке
3. Запускает reduce-горутину, которая:
   - Собирает промежуточные суммы от map-горутин
   - Вычисляет общую сумму квадратов
4. Выводит общую сумму

При этом тебе нужно:
- Использовать два `WaitGroup`: для `map` и `reduce` этапов
- Правильно закрывать каналы после отправки данных
- Обрабатывать случаи с пустым вводом или с количеством элементов, не делящимся на `3`
- Выводить только итоговую сумму

Ограничения:
- Не использовать глобальные переменные
- Размер чанка определять как `len(data)/3`
- Для map использовать буферизированный канал (размер `3`)
- Для reduce использовать небуферизированный канал
