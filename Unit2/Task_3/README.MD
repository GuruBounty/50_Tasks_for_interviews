#### Задача 3: Конвейер данных

Нам дано:
- Срез целых чисел

Напиши программу, которая:
1. Создает конвейер из 3 этапов:
   - a. Генератор: отправляет числа из среза в канал
   - b. Множитель: получает числа, умножает на 2 и отправляет дальше
   - c. Вывод: получает числа, добавляет 10 и выводит `"Результат: <число>"`
2. Каждый этап работает в отдельной горутине
3. Использует буферизированные каналы
4. Корректно закрывает каналы после обработки всех данных
5. Ожидает завершения всех горутин

При этом тебе нужно:
- Использовать `WaitGroup` для синхронизации
- Закрывать каналы на этапе генерации
- Обрабатывать числа в порядке генерации
- Формат вывода: `"Результат: <число>"`

Ограничения:
- Не использовать глобальные переменные
- Каждый этап в отдельной горутине
- Размер буфера каналов: 10
