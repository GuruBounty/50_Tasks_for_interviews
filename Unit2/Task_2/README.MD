### Задача 2: WaitGroup
Нам дано:
- Число N (количество горутин)
- Время работы каждой горутины в миллисекундах

Напиши программу, которая:
1. Запускает N горутин, каждая из которых:
   - Выводит `"Горутина [i] начала работу"`
   - Ожидает указанное время (`time.Sleep`)
   - Выводит `"Горутина [i] завершила работу"`
2. Использует `sync.WaitGroup` для ожидания завершения всех горутин
3. После завершения всех горутин выводит `"Все горутины завершили работу"`

При этом тебе нужно:
- Использовать указатель на `WaitGroup` при передаче в горутины
- Корректно вызывать `Add()`, `Done()` и `Wait()`
- Гарантировать вывод финального сообщения только после всех горутин
- Обрабатывать случай `N ≤ 0`

Ограничения:
- Не использовать глобальные переменные
- Не использовать `time.Sleep` в `S` для ожидания
- Четко следовать формату вывода
