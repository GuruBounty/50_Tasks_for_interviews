#### Задача 8: Fan-in — слияние 3 каналов

**Краткое описание и цель навыка**  

Нам дано:** три входных канала** `chan int`, в каждом приходит поток чисел (каналы закрываются когда данных больше нет).

Нужно: написать функцию `fanIn(a, b, c <-chan int) <-chan int`, которая возвращает **один выходной канал**, из которого можно прочитать все значения из `a`, `b`, `c` (в любом порядке).

Чтобы сделать — вспомни: горутины, `WaitGroup`, `range` по каналу, и самое главное — b.

**Условие**

Нам дано: 3 канала `a`, `b`, `c` типа `int`.  
Напиши: функцию fanIn(a, b, c  <-chan int) <-chan int, которая:
- запускает “читателей” для каждого входного канала,
- пересылает все значения в общий канал out,
- закрывает out, когда все три входных канала закрылись и все значения пересланы.
  
При этом нельзя:  
- закрыть out слишком рано (иначе паника/потеря данных),
- допустить дедлок (например, если никто не читает out или wg не дождались),
- писать в закрытый out.

Нужно:
- корректно завершать все горутины,
- гарантировать, что out будет закрыт ровно один раз.
