#### Задача 5: "Таймауты обработки запросов"
**Описание задачи:**  
Вы разрабатываете систему обработки запросов с ограничением времени выполнения. Программа должна:
 1. Получить два числа: N (количество запросов) и timeout (максимальное время обработки в миллисекундах)
 2. Для каждого запроса (от 1 до N):
    - Запустить обработку в отдельной горутине
    - Обработка занимает случайное время (0-200% от timeout)
    - Если обработка завершается за timeout - вернуть результат (запрос * 10)
    - Если время превышено - вернуть `"timeout"`

 3. Вывести результаты в порядке завершения обработки

**Условия**:  
Использовать `select с time.After`  
Каждый запрос обрабатывается в своей горутине  
Порядок вывода результатов может не соответствовать порядку запросов

|Input|Output|
|:---|:---|
|3 60|Число 1: 2 <br>Число 2: 4 <br> Число 3: timeout|
