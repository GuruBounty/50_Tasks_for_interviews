### Задача 1: Параллельный поиск
Нам дан:
- Срез целых чисел (до 1000 элементов)
- Целевое число для поиска

Напиши функцию parallelSearch(), которая:
1. Делит срез на две примерно равные части
2. Запускает две горутины для поиска числа в каждой части
3. Возвращает индекс первого вхождения числа ИЛИ -1 если число не найдено
4. Если число найдено в обеих частях, возвращает меньший индекс

При этом тебе нужно:
- Использовать канал для получения результатов из горутин
- Использовать sync.WaitGroup для ожидания завершения горутин
- Закрывать каналы после завершения работы
- Обрабатывать случай пустого среза

Ограничения:
- Не использовать глобальные переменные
- Не использовать пакеты для параллельного поиска (только базовые примитивы)
