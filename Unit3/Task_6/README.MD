#### Задача 6 — Упрощённый CSV-парсер (state machine)

**Краткое описание**

Нам дана одна строка — это “упрощённый CSV”: поля разделены запятыми `,`, поля могут быть в двойных кавычках `"..."`.
Внутри кавычек запятая **не разделитель**. Экранирования нет (то есть \" не поддерживаем).

Нужно распарсить строку на поля и вывести их построчно.

**Условие**

**Нам дано**: строка `line` — одна CSV-строка.
- разделитель: `,`
- кавычки: `"` (двойные)
- **экранирования нет**
- кавычки используются только как “обрамление поля”
- допускаются пробелы и любые Unicode-символы

**Напиши:** программу, которая разбивает строку на поля и выводит:
- на каждой строке — одно поле (после разбора)
- внешние кавычки у поля нужно убрать

Пример: `"a,b"` → `a,b`
