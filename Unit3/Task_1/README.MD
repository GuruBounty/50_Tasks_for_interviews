#### Задача 1 — Reverse строки по рунам (Unicode-safe)

**Описание**

В Go строки хранятся в кодировке **UTF-8**, поэтому один “символ”, который мы видим, может занимать несколько байт.  
Если разворачивать строку побайтово, Unicode-символы (кириллица, emoji) будут повреждены.

В этой задаче тебе нужно написать **Unicode-safe** реверс строки.

**Условие**

Нам дана: одна строка s в кодировке **UTF-8.**  
Строка может содержать:
- латиницу
- кириллицу
- пробелы
- emoji и другие Unicode-символы

**Напиши**: программу, которая выводит строку `s`, развернутую **по рунам** (Unicode code points).

**При этом запрещено:**
- конвертировать строку в []byte и реверсить байты
- итерироваться по строке как по массиву байт (`s[i]`)

**Нужно:**
- корректно обработать любые Unicode-символы
- сохранить все символы строки, включая пробелы и знаки пунктуации
