#### Задача 3 — Самая длинная подстрока без повторов (Unicode-safe)

**Краткое описание**

Нам дана строка `s` (UTF-8, может содержать кириллицу/emoji/пробелы). Нужно найти **длину** самой длинной подстроки без повторяющихся **рун**.

**Цель (навык)**
- **sliding window** (скользящее окно)
- хранение “последней позиции” через `map[rune]int`
- работа **по рунам**, а не по байтам

**Условие**

**Нам дано:** одна строка `s`.  
**Напиши**: программу, которая выводит длину (в рунах) самой длинной подстроки без повторов.  
**При этом нельзя:** решать побайтово (через `[]byte` и индексы байт), иначе Unicode сломается.
**Нужно**: решить за `O(n)` по количеству рун (через sliding window).
