#### Задача 7 — Нормализация пробелов (Unicode-safe)

**Краткое описание**

Нам дана строка `s` (UTF-8). В строке могут встречаться любые **пробельные символы Unicode**: пробелы, табы, переводы строки и т.д.
Нужно нормализовать их так, чтобы **любая последовательность пробельных символов превратилась в один обычный пробел ' '**.

Пример:
`"a b \n c" → "a b c"`

**Условие**

**Нам дано:** строка `s`.  
**Напиши**: программу, которая:
- заменяет **любую последовательность пробельных символов** на **один пробел**
- сохраняет порядок всех остальных символов

**При этом нельзя/нужно:**
- нужно использовать `unicode.IsSpace`
- **нельзя** считать пробелом только `' '` (ASCII), т.к. есть `\n`, `\t`, NBSP и т.д.
- **не нужно** удалять пробелы полностью — они схлопываются в один
