#### Задача 2 Удалить дубликаты (preserve order)

**Условие** 

**Нам дано:** срез строк длины `n`.  
**Напиши:** функцию, которая удалит повторяющиеся строки, **сохранив порядок первых появлений**.
**При этом нужно:** сделать это эффективно по времени — без вложенных циклов O(n²).
**При этом нельзя:** сортировать массив (порядок важен).

**Формат ввода/вывода**

**Ввод**:
- Первая строка: `n`
- Вторая строка: `n` строк через пробел

**Вывод**  
Выведите результат (уникальные строки в порядке первых появлений) через пробел.
Если результат пустой — пустую строку.
