#### Задача 1 Удаление элемента из среза по индексу

**Условие**

**Нам дано:**

срез строк `a` длины n и индекс `k`.

**Напиши:** 

функцию, которая удаляет элемент с индексом k и возвращает новый срез длины `n-1`.

**При этом нужно:**
- сохранить порядок остальных элементов
- корректно обработать крайние случаи

**При этом нельзя:**
- бездумно создавать новый срез через append “на всякий случай”

**Формат ввода**
`n`  
`a[0] a[1] ... a[n-1]`  
`k`

**Формат вывода**

Выведите элементы результирующего среза через пробел.
Если результат пустой — выведите пустую строку.
