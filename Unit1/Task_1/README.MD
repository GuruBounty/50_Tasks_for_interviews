#### Задача 1: Палиндром чисел

Нам дано: целое число (положительное или отрицательное)
Напиши код в функции `isNumPalindrome`, чтобы на выходе получалось:
- `true` если число читается одинаково слева направо и справа налево (`121` → `true`)
- `false` если не палиндром (`123` → `false`) или отрицательное (-`121` → `false`)

При этом тебе нельзя:
- Преобразовывать число в строку (запрещено использовать strconv, fmt.Sprint и т.д.)
- Использовать массивы/срезы для хранения цифр

Чтобы сделать эту задачу, вспомни:
- Как извлекать последнюю цифру числа (оператор `%`)
- Как удалять последнюю цифру (оператор `/`)
- Как собирать число из цифр в обратном порядке
