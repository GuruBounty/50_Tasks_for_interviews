#### Задача 6: Сумма цифр числа (без строк)

**📘 Условие задачи**  
Нам дано одно целое число `n`.  
Твоя задача — реализовать функцию:
- `func sumDigits(n int) int`

Она должна вернуть **сумму всех цифр числа**.

**⚠️ Важные условия**  
- Число `n` м**ожет быть отрицательным**
- **Нельзя** использовать:
    - строки (string)
    - преобразование числа в строку
- Нужно работать только с **арифметикой**

Пример:

`n = -1203 сумма = 1 + 2 + 0 + 3 = 6`
