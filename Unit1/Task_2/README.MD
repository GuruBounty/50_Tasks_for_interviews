### Задача 2: Конвертер температуры

Нам дано: значение температуры типа any (может быть float64, string или другой тип)
Напиши код в функции FahrenheitToCelsius, чтобы на выходе получалось:
- Температура в Цельсиях (float64) с точностью до 2 знаков при корректном вводе
- Ошибка (error) при некорректном типе данных с подписью ("invalid type")

При этом тебе нужно:
- Использовать именованные возвраты (c float64, err error)
- Проверять тип данных через type switch
- Для расчета: C = (F - 32) * 5/9

Чтобы сделать эту задачу, вспомни:
- Как работать с типом any (type assertion)
- Как создавать ошибки: errors.New() или fmt.Errorf()
- Как использовать type switch для проверки типов
