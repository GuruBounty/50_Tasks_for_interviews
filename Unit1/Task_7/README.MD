### Задача 7: Проверка високосного года
**📘 Условие задачи**  
Нам дано одно целое число — год.  
Твоя задача — реализовать функцию: `func isLeap(year int) bool`  
Функция должна вернуть true, если год является **високосным**, и false — в противном случае.

**📜 Правило високосного года**
Год является високосным, если:

1. Он делится на 400 → високосный

2. Иначе, если он делится на 100 → не високосный

3. Иначе, если он делится на 4 → високосный

4. Во всех остальных случаях → не високосный

**⚠️ Порядок проверок важен.**
