#### Задача 3: Подсчет гласных

Нам дано: строка произвольных символов (включая русские буквы)
Напиши код в функции `countVowels`, чтобы на выходе получалось:
- Целое число - количество гласных букв (английских и русских)
- Регистр не должен влиять (А и а считаются одинаково)

При этом тебе нужно:
- Использовать `map[rune]bool` для хранения гласных (как в шаблоне ниже)
- Приводить символы к нижнему регистру `(unicode.ToLower)`
- Итерироваться по рунам (используя range по строке), а не байтам

Чтобы сделать эту задачу, вспомни:
- Как работать с Unicode в Go (тип rune)
- Как использовать map для проверки принадлежности: *if vowels[r] { ... }*
- Как конвертировать символ в нижний регистр: `unicode.ToLower(r)`
