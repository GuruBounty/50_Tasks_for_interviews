#### Задача 5: Максимум из трёх (без math)
📘 **Условие задачи**
Нам даны три целых числа `a`, `b`, `c`.  
Твоя задача — реализовать функцию:  
`func maxOfThree(a, b, c int) int`  
Она должна вернуть максимальное из трёх чисел.

✅ **Требования**
Чтобы задача считалась решённой правильно: 
- нельзя использовать пакет math
- нельзя использовать массивы/срезы
- нужно решить задачу только через `if/else` и сравнения `>` / `<`
-  функция должна корректно работать, если:
    - числа отрицательные
    - есть равные значения (например, 5 5 4)
    - все три одинаковые

🧠 **Ключевая идея задачи**
Эта задача тренирует базовую “мышцу” Go: 
- чистая логика через if
- аккуратные сравнения
- минимум кода без “магии”

На собеседованиях часто смотрят, умеешь ли ты не запутаться в простых условиях.

**🧪 Формат ввода и вывода **  
**Ввод**  
Три целых числа `a`, `b`, `c`.  
Пример:
```
3 7 5
```
**Вывод**  
Одно целое число — максимум.  
Пример:
```
7
```
